# Tamil Calendar WhatsApp Bot

A WhatsApp chatbot that returns Tamil daily panchangam for:
- Today
- Tomorrow
- Yesterday

Hosted on Render (free).

## Run locally:
pip install -r requirements.txt
python whatsapp_bot.py

## Deploy:
Push to GitHub → Render → New Web Service
Set environment variables:
- WHATSAPP_TOKEN
- PHONE_NUMBER_ID
